# td_ShotSetup

Fast and consistent shot setup tool for Nuke based on an input plate.

## Features

- Save and version up scripts
- Match frame range, format, and FPS from input
- Set color management and OCIO configuration
- Manual control over project settings when needed

## Usage

Connect a Read node and use the available tools as needed.
